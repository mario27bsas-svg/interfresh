Guía de Integración para WordPress & Elementor

Este archivo HTML es de empaquetado directo de código. Para incrustar este cotizador premium en su sitio web de WordPress utilizando Elementor, siga estos sencillos pasos:

  1. 1
    Copie el Código Fuente: Abra o edite el archivo wp-elementor-integration.html, seleccione todo el documento y cópielo.
  2. 2
    Arrastre Widget "HTML" en Elementor: Ingrese al panel editor de Elementor dentro de su página WordPress, busque el widget "HTML" (o "HTML personalizado") y arrástrelo a su sección preferida.
  3. 3
    Pegue el Código Copiado: En la configuración del widget HTML personalizado, pegue el bloque de código copiado entero. ¡Elementor interpretará en tiempo real el diseño y cargará los fletes e íconos interactivos de forma instantánea!
